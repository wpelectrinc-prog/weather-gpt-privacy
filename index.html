🌤️ 台灣天氣小幫手 隱私權政策

📅 最後更新日期：2024年9月24日
簡要說明：本服務僅查詢天氣資訊，不收集個人身份資料，所有資料來源為中央氣象署公開資料。

1. 服務說明

台灣天氣小幫手是一個基於 OpenAI ChatGPT 技術的天氣查詢服務，透過中華民國交通部中央氣象署開放資料 API 提供台灣地區的即時天氣預報資訊。

2. 資料收集範圍

本服務僅收集以下必要資訊：

  • 查詢地點：您輸入的縣市或鄉鎮市區名稱
  • 查詢時間：您指定的時間範圍（今日、明日、一週等）
  • 對話記錄：ChatGPT 平台依據其政策自動記錄的對話內容

我們不會收集：姓名、電話、地址、電子郵件等個人身份資訊。

3. 資料使用目的

收集的資料僅用於以下目的：

  • 提供準確的天氣預報查詢服務
  • 改善服務品質和使用者體驗
  • 遵循 OpenAI ChatGPT 平台的使用條款和政策

4. 資料來源聲明

所有天氣預報資料來源為：

  • 中華民國交通部中央氣象署開放資料平台
  • 資料網址：https://opendata.cwa.gov.tw

我們不儲存、修改或聲稱擁有任何氣象資料的著作權。所有預報資料的準確性和時效性以中央氣象署官方發布為準。

5. 資料保護措施

我們承諾：

  • ✅ 不儲存使用者的個人身份資訊
  • ✅ 不將查詢資料用於任何商業用途
  • ✅ 不與第三方分享使用者查詢記錄
  • ✅ 嚴格遵循 OpenAI ChatGPT 的隱私權保護政策

6. 第三方服務聲明

本服務整合以下第三方平台和服務：

  • OpenAI ChatGPT：AI 對話平台
    隱私權政策：https://openai.com/privacy
  • 中央氣象署開放資料 API：天氣資料來源
    使用條款：https://opendata.cwa.gov.tw/dist/opendata-swagger.html

這些第三方服務各有其隱私權政策和使用條款，建議您一併了解。

7. 資料保留政策

  • 我們不主動保留任何使用者查詢資料
  • 所有對話記錄的保留和刪除依照 OpenAI ChatGPT 平台政策處理
  • 天氣資料即時從中央氣象署 API 取得，不進行本地儲存

8. 使用者權利

作為使用者，您擁有以下權利：

  • 🔄 隨時停止使用本服務，無需任何理由
  • 🗑️ 要求刪除對話記錄（透過 ChatGPT 平台功能）
  • 📋 了解資料使用情況，隨時查閱本隱私權政策
  • ❓ 提出隱私權相關問題，我們將盡力回應

9. 免責聲明

  • 天氣預報資料的準確性以中央氣象署官方發布為準
  • 本服務僅供參考，重要決策請以官方氣象資訊為主
  • 我們不對因使用本服務而產生的任何損失承擔責任

10. 聯絡方式

如有任何隱私權相關問題或建議，請透過以下方式聯繫：

  • 透過 ChatGPT 對話介面直接詢問
  • 或在使用本服務時提出相關問題

11. 政策更新

本隱私權政策可能會因應法規變更或服務改善而不定期更新。更新時會修改頁面頂部的「最後更新日期」。建議您定期查看最新版本。

12. 適用法律

本隱私權政策適用中華民國法律，包括但不限於：

  • 個人資料保護法
  • 消費者保護法
  • 其他相關法規
